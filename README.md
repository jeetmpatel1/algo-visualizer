# algo-visualizer
https://jeetmpatel1.github.io/algo-visualizer/

This project is an interactive side-by-side visualization of Container With Most Water, Trapping Rain Water, and Largest Rectangle in Histogram — with animated two-pointer movement, stack operations, and step-by- step explanations.


### Trapping rain water
  - [Leetcode link](https://leetcode.com/problems/trapping-rain-water/description/)
  - **No space betweeen buildings**
  - how many units of water each building can hold without spilling on either side.
### Max rectangle area histogram
  - [Leetcode link](https://leetcode.com/problems/trapping-rain-water/description/)
  - **No space between histogram bars**
  - how much you can extend each bar on left and right side creates historgram
### Container With Most Water
  - [Leetcode link](https://leetcode.com/problems/container-with-most-water/)
  - **clear space between container** - actually that's where the water is
  - how big container you can find out
